# Trim hatch script
Simple script for trimming hatches in Rhino 6.0

__Installation__:
1. git clone or download zip
2. In rhino go _Tools-PythonScript-Run_... select _trim_hatches.py_ script

__Using__:

Script asks you to select hatch first then trimming line after that it will start built-in _Trim_ command and let you choose parts to be trimmed. Voil√†.

Script uses python API
